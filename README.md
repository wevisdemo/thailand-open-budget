## Usages

Requirements: Node.js 18+, npm

### Local dev server

Install packages

```sh
npm i
```

Run dev server

```sh
npm run dev
```

### Build

```sh
npm run build
```

Output will be in the `out/` folder (static files, ready to deploy).
